# HR Analytics Dashboard

An end-to-end HR analytics project built using Odoo ERP and Power BI, developed during my internship at Arth Consultancy Services.

## Overview

This project simulates a real-world HR analytics workflow — from inserting and managing structured employee data in an ERP system to building interactive dashboards that track key HR KPIs. The goal was to surface actionable insights across core HR functions including recruitment, attendance, skills, and leave management.

## Tools & Technologies

- **Odoo ERP** — Data entry and management across HR modules
- **Power BI** — Interactive dashboard development
- **MS Excel** — Data cleaning and transformation

## Project Workflow

### 1. Data Setup in Odoo ERP
Inserted and managed structured mock HR data across 5 modules:
- **Recruitment** — Job positions, applications, and hiring pipeline
- **Employees** — Employee profiles and department structure
- **Skill Management** — Employee skills and skill levels
- **Attendance** — Daily attendance records (~1,000+ entries for ~50 employees)
- **Time-Off** — Leave requests and approval status

### 2. Data Export & Cleaning
- Exported module-wise data from Odoo ERP
- Cleaned and transformed data using MS Excel for consistency and analysis readiness

### 3. Power BI Dashboard
Built interactive dashboards tracking the following KPIs:
- Headcount by department
- Attendance trends and absenteeism rate
- Skill distribution across the organization
- Time-off patterns and leave utilization
- Recruitment pipeline and funnel drop-offs

## Key Insights Covered

- Which departments had the highest absenteeism
- Skill gaps across teams
- Leave utilization patterns by employee type
- Recruitment funnel efficiency
